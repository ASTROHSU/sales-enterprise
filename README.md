# 區塊勢 | 金融業企業訂閱方案

這是一個使用 React 和 Tailwind CSS 建立的簡報式網頁，展示區塊勢企業訂閱方案的內容。

## 功能特色

- 📱 響應式設計，支援桌面與行動裝置
- 🎨 現代化的 UI 設計，使用 Tailwind CSS
- ⚡ 使用 React 18 建立互動式簡報
- ⌨️ 支援鍵盤操作（方向鍵、空白鍵）
- 🎯 簡潔的簡報式導覽體驗

## 技術棧

- React 18
- Tailwind CSS
- Babel Standalone（用於瀏覽器端 JSX 轉譯）

## 使用方式

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 或使用本地伺服器：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 或使用 Node.js
   npx serve .
   ```

## 操作說明

- **下一頁**：點擊右側箭頭按鈕、按鍵盤右方向鍵、空白鍵或 Enter
- **上一頁**：點擊左側箭頭按鈕、按鍵盤左方向鍵
- **跳轉頁面**：點擊底部進度條的圓點

## 授權
